# CiteHue 🎨

> **Aesthetic color palettes for Computer Science top-tier conferences & journals.**

**CiteHue** is a curated collection of color schemes specifically designed for researchers in fields like Computer Vision (CVPR), Machine Learning (NeurIPS), and Graphics (SIGGRAPH). Built with "vibe coding" philosophy, it balances the precision of academic charts with the aesthetics of modern UI design.

## ✨ Features

- **Double Aesthetics**: Choose between **Apple-inspired** (fluid, minimalist) and **Google-inspired** (Material Design, structured) interfaces.
- **Eye-Friendly**: Zero use of pure black (`#000000`) or pure white (`#FFFFFF`) to ensure visual comfort and print-friendly results.
- **Domain Optimized**:
  - `NeurIPS Deep Learning`: High-tech teals and deep purples.
  - `CVPR Object Detection`: Vibrant, high-contrast markers.
  - `ACM SIGGRAPH`: Creative gradients for rendering.
  - `IEEE Trust`: Formal and reliable slate-blue tones.
- **Instant Copy**: One-click to copy Hex codes directly to your Matplotlib, Seaborn, or PGFPlots code.

## 🚀 Quick Start

Simply clone the repository and open the style you prefer in your browser:

```bash
# To view Apple style
open apple-palette/index.html

# To view Google style
open google-palette/index.html
```

## 📂 Project Structure

- `apple-palette/`: Minimalist, SF-font driven interface.
- `google-palette/`: Material Design driven interface with Google Sans.
